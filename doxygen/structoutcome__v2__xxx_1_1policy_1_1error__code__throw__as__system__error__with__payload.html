Detailed Description

template<class R, class S, class P>
struct outcome_v2_xxx::policy::error_code_throw_as_system_error_with_payload< R, S, P >

Policy interpreting S as a type implementing the std::error_code contract, P as a payload type, and any wide attempt to access the successful state calls an ADL discovered free function throw_as_system_error_with_payload().

Can be used in outcome only.
