Detailed Description

template<class EC>
struct outcome_v2_xxx::policy::error_enum_throw_as_system_error< EC >

Policy interpreting EC as an enum convertible into the std::error_code contract and any wide attempt to access the successful state throws the error_code wrapped into a std::system_error
