Used to return from functions (i) a value and (a positive status and/or a payload) or (ii) no value and (a negative status and/or a payload). constexpr capable.

Parameters:

  • R - The optional type of the successful result (use void to disable).

  • S - The optional type of the status result (use void to disable). Must be either void or DefaultConstructible.

  • P - The optional type of the payload/exception result (use void to disable). Must be either void or DefaultConstructible.

  • NoValuePolicy - Policy on how to interpret types S and P when a wide observation of a not present value occurs.
